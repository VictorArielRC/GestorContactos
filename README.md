# GestorContactos

Aplicación WPF en **C#/.NET 8** para la gestión de contactos, utilizando **SQL Server** como base de datos y siguiendo una arquitectura en capas.

## 🧱 Arquitectura

- **Dominio** → Entidades, validadores y contratos (interfaces).
- **Datos** → Entity Framework Core, DbContext y repositorios.
- **Servicios** → Lógica de negocio y orquestación.
- **WPF (UI)** → Interfaz gráfica con MVVM y ventanas/dialogos.

## 🚀 Tecnologías utilizadas

- .NET 8 LTS  
- WPF (Windows Presentation Foundation)  
- Entity Framework Core  
- SQL Server LocalDB  
- Git + GitHub  

## ⚙️ Funcionalidades previstas

- CRUD de clientes (crear, leer, actualizar, eliminar).  
- Validación de RUT chileno.  
- Filtros y búsqueda avanzada.  
- Paginación de resultados.  
- Exportación de datos (Excel/PDF).
